Global Resource Loader
-------------------------------------------------------------------------------

Version: 1.0.5
Author: Rowan Lewis <me@rowanlewis.com>
Build Date: 3 March 2011
Requirements: Symphony 2.2


Installation
-------------------------------------------------------------------------------

1.  Upload the 'globalresourceloader' folder in this archive to your Symphony
  'extensions' folder.

2.  Enable it by selecting the "Global Resource Loader", choose Enable from the
  with-selected menu, then click Apply.

4. View the Blueprints > Globals page to make changes.
